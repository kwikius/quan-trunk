Quan STM32

The quan/stm32/ library is written in C++11 to simplify and clarify use of the high performance STM32 architecture without compromising performance.

The following link shows a simple application using the quan::stm32 library to blink an LED forever... ../stm32f4/example1.html

  • stm32::Gpio - Archetype representing the STM32 General Purpose IO port modules
  • stm32::gpio::Pin - Archetype representing a stm32::Gpio port pin
  • stm32::PeriphReg - Archetype representing a module register
  • stm32::Usart - Usart Archetype
  • Tim - Timer Archetype
  • Dma - DMA Concept
  • Nvic - Interrupt Concept
