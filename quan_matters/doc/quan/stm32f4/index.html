Quan STM32f4

The quan/stm32f4/ library is written in C++11 to simplify and clarify use of the high performance STM32F4 architecture without compromising performance.

The following link shows a simple application using the quan::stm32f4 library to blink an LED forever... ./example1.html

  • Gpio - Archetype representing the STM32F4 General Purpose IO port modules
  • gpio::Pin - Archetype representing a Gpio port pin
  • PeriphReg - Archetype representing a module register
  • Usart - Usart Archetype
  • Tim - Timer Archetype
  • Dma - DMA Concept
  • Nvic - Interrupt Concept
