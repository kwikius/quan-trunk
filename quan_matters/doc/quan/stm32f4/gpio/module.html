quan::stm32f4::gpio::module<address>

defined in <quan/stm32f4/gpio/module.hpp>

model of Gpio

where address is a uint32_t whose value is the memory address of the module.

The various registers composing the module are represented as member typedefs and as member objects. The structure is not copyable or instantiable directly. (logically it already exists as a singleton at a particular address), but a pointer to it can be obtained using the static memmber get() function.

The IO ports GPIOA to GPIOI are represented as module typedefs in the quan::stm32f4 namespace
