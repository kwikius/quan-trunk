The quan source code is now available on Github

I am pleased to announce that the full Quan source is now available on Github.

The repository can be viewed at https://github.com/kwikius/quan-trunk

Download the repo from https://github.com/kwikius/quan-trunk/archive/master.zip

(You can also download the sources as a zip, by hitting the ZIP button top left on the Github repo page or clone the repo using a git client.)

Note that the code is written in C++11 using gcc. If using gcc you will need gcc4.7 and higher to get the C++11 functionality (using the -std=c++11 switch))
